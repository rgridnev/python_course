Модель, описывающая зависимость оценок качества вина от его лабораторных метрик на базе датасета https://www.kaggle.com/rajyellow46/wine-quality

В директории содержится:
1. Датасет data.csv
2. Блокнот с кодом wine_model.ipynb
3. pickle-дамп модели wine_model.pkl
4. отчет об исследовании Определение качества вина по его объективным характеристикам.pdf
